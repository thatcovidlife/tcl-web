# Product Context

## Project Overview

This is a Nuxt.js application called "That Covid Life" that serves as a comprehensive platform for COVID-19 related content, resources, and community features. The platform provides users with access to articles, videos, research, and community discussions related to COVID-19.

## Purpose

The platform addresses the need for reliable, accessible COVID-19 information and community support during the pandemic. It serves as a centralized hub for:

- COVID-19 research and publications
- Community discussions and support
- Resource sharing and discovery
- Multi-language accessibility for global users

## Key Features

- **Content Management**: Sanity-powered CMS for managing articles, videos, and research publications
- **User Authentication**: Complete user profile management and authentication system
- **RSS Feed Generation**: Automated RSS feeds for content syndication
- **Multi-language Support**: Internationalization (i18n) with support for English, Spanish, French, and Portuguese
- **Resource Directory**: Organized collection of COVID-19 related resources
- **Search & Discovery**: Advanced search functionality and content filtering
- **Community Features**: User contributions, bookmarks, and support requests
- **Mobile Responsive**: Optimized for mobile devices with dedicated mobile navigation
- **Performance Monitoring**: Comprehensive error tracking and performance monitoring with Sentry

## Technology Stack

- **Frontend**: Nuxt.js (Vue.js framework with SSR)
- **Content Management**: Sanity headless CMS
- **Database**: Drizzle ORM for type-safe database operations
- **Error Tracking**: Sentry for error tracking and performance monitoring
- **Styling**: Tailwind CSS utility-first framework
- **Language**: TypeScript for type safety
- **State Management**: Pinia for centralized state management
- **Internationalization**: Vue I18n for multi-language support
